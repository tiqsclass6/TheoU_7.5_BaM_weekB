## 🛠️ **Troubleshooting Techniques**

If you encounter issues while working on Killercoda labs:

1. **Refresh the Environment** – Sometimes the terminal may freeze; restarting the lab helps.  
2. **Check Command Syntax** – Typos in Linux commands are a common cause of errors. Use `man <command>` for help.  
3. **Networking Issues** – If `ping` or `curl` fails, verify you are in the correct container environment.  
4. **Re-run Exercises** – If screenshots are missing or corrupted, redo the exercise and capture again.  
5. **GitHub Uploads** – Ensure you commit with:

   ```bash
   git remote add origin https://github.com/tiqsclass6/TheoU_7.5_BaM_weekB.git
   git branch -M main
   git add .
   git commit -m "initial commit"
   git push -u origin main
   ```

   > to keep the repo updated.
